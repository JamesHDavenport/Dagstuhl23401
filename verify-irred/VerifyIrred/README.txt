Various files connected to JHD's project at Dagstuhl 23401 (https://drops.dagstuhl.de/entities/document/10.4230/DagRep.13.10.1) and generalisations
SparsePoly is a computational implementation of univariate polynomials begun by Mario Carniero at Hausdorff (https://www.mathematics.uni-bonn.de/him/programs/current-trimester-program/him-trimester-program-prospects-of-formal-mathematics)
MySparsePoly is an attempt (based on SparsePoly) to do sparse distributed polynomials for Goeber Base applications
MySparsePolyv2 is a second attempt with better typing
